# Greedy

- **정리**
  - 현재 상황에서 가장 최선만 선택한다.
- **이런것들을 고려해야 한다**
  - 현재에서 최선의 선택만 계속해도 문제가 안되는가?
  - 항상 최적의 해가 나오지 않을 수 있다.
  - 현재 상태에서 최적의 길만 선택하더라도 정당한지? 에 대한 정당성을 검증해야한다
- **특징**
  - 특별한 알고리즘 개념이나 이론이 필요 없다
    - 하지만 뒤에 다익스트라는 그리디로 분류됨
  - 창의력을 물어보는 문제다
